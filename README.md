# 25Spring-Optimization-Operations-Research
This repository is all about the course `Optimization and Operations Research` resources contained in the whole teaching schedule on 2025 Spring semester in Haide College, Ocean University of China.
Optimization and Operations research is a course that introduces how to do a Simplex method and analyse the result of it, which also cares about some simple heuristic algorithms to solve complex optimization problems.
This repository is only for reference usage for any students who are interested in this course, please strictly follow the academic integrity polices to avoid any concerns.
本仓库仅用于给希望了解这一课程的学生提供参考，请严格遵守所在学校的学术诚信守则以避免任何问题。
